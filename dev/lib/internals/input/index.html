Scats.internal.input — Module

Module containing a type for storage and interaction with input data.

source
Scats.internal.input.InputStruct — Type
InputStruct()

Instantiate this type to interact with input data.

Data

  • N::IT=0: sample size;
  • Δt::RT=0: sample step;
  • q::RT=0: significance level;
  • t::Vector{RT}=[]: time array;
  • x::Vector{RT}=[]: values array.

Methods

  • read!(file::AbstractString): read input data from a file;
  • write(file::AbstractString): write input data to a file;
  • example(file::AbstractString): generate an example of the input/output file;
  • reset!(): reset an instance to default values.

Note

Data can be also read calling an instance like so:

using Scats
s = Scats.api()
file, _ = mktemp()
s.input.example(file)
s.input(file)
