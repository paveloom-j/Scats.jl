Scats.internal.gen — Module

Module containing a type for interaction with the time series generator.

source
Scats.internal.gen.GenStruct — Type
GenStruct()

Instantiate this type to interact with the time series generator.

Data

  • N::IT=0: sample size;
  • Δt::RT=0: sample step;
  • q::RT=0: significance level;
  • α::RT=0: parameter α of a linear trend;
  • β::RT=0: parameter β of a linear trend;
  • r::IT=0: number of harmonics;
  • A::Vector{RT}=[]: amplitudes array;
  • ν::Vector{RT}=[]: frequencies array;
  • ϕ::Vector{RT}=[]: phase shifts array;
  • γ::RT=0: «signal-to-noise» ratio.

Methods

  • read!(file::AbstractString): read generator parameters from a file;
  • example(file::AbstractString): generate an example of a file containing the generator parameters;
  • gen(): generate time series;
  • reset!(): reset an instance to default values.

Note

Data can be also read calling an instance like so:

using Scats
s = Scats.api()
file, _ = mktemp()
s.gen.example(file)
s.gen(file)
