println(io::IO, array::Array{RT, 1})

Print an array containing RT values.

Usage

using Scats: internal.prec.RT, internal.extras.println
x = Array{RT}([1.0, 2.0, 3.0])
println(x)

Note: this doctest requires x64 architecture.
