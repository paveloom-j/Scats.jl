Result

Scats.Internal.Result — Module

Module containing the type to interact with result data.

source
Scats.Internal.Result.ResultStruct — Type
ResultStruct()

Instantiate this type to interact with result data.

Data

  • Δt::RT=0: sample step;
  • t::Vector{RT}=[]: time array;
  • x::Vector{RT}=[]: values array;
  • q::RT=0: significance level;
  • threshold::RT=0: signal threshold;
  • X_FFT_ABS::Vector{RT}=[]: values array;
  • ν::Vector{RT}=[]: periodogram frequencies array;
  • D::Vector{RT}=[]: periodogram values array;
  • c::Vector{RT}=[]: correlogram values array;
  • cw::Vector{RT}=[]: weighted correlogram values array;
  • Dw::Vector{RT}=[]: smoothed periodogram values array.

Methods

  • reset!(): reset an instance to default values.

reset!(Result::ResultStruct)

Reset an instance of ResultStruct to default values.

Usage

using Scats
s = Scats.API()
file, _ = mktemp()
s.gen_example(file)
s.Gen(file)
s.gen!()
# Generate result data
s.Result.reset!()
